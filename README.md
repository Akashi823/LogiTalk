# LogiTalk

**LogiTalk** — це простий месенджер, схожий на Viber або Telegram, який працює в локальній мережі. За його допомогою можна спілкуватися, надсилати зображення та зберігати історію повідомлень.

## Основні можливості

* 💬 Надсилання текстових повідомлень
* 🖼️ Надсилання зображень
* 😊 Автоматична заміна текстових смайликів на емодзі
* 🎨 Зміна теми оформлення
* 💾 Збереження історії чату
* 🧹 Очищення чату
* 👤 Зміна імені користувача
* 🟢 Відображення статусу підключення до сервера

> **Увага!** Програма працює лише тоді, коли запущений сервер.

## Що потрібно для роботи

Перед запуском переконайтеся, що у вас є:

* Python 3.10 або новішої версії;
* встановлені необхідні бібліотеки;
* запущений сервер чату.

## Встановлення

### 1. Встановіть Python

Завантажте Python з офіційного сайту:

https://www.python.org/downloads/

Під час встановлення поставте галочку **Add Python to PATH**.

### 2. Встановіть бібліотеки

Відкрийте командний рядок або Terminal та виконайте команду:

bash
pip install customtkinter pillow

### 3. Запустіть сервер

Спочатку необхідно запустити сервер чату.

Після запуску сервера можна відкривати програму.

## Як користуватися програмою

### Надсилання повідомлення

1. Введіть текст у поле введення.
2. Натисніть кнопку **>**.
3. Повідомлення з'явиться в чаті.

### Надсилання зображення

1. Натисніть кнопку **📂**.
2. Виберіть потрібне зображення.
3. Воно автоматично відправиться всім користувачам.

### Відкриття меню

Натисніть кнопку **▶️** у верхньому лівому куті.

У меню доступні такі функції:
<img width="732" height="799" alt="Screenshot 2026-07-18 141014" src="https://github.com/user-attachments/assets/99027c39-3bc4-445d-b7c3-dcb1b5223c05" />

* зміна імені користувача;
* збереження історії чату;
* очищення чату;
* зміна теми оформлення.

Щоб закрити меню, натисніть кнопку **◀️**.

## Зміна імені

1. Відкрийте меню.
2. Введіть новий нікнейм.
3. Натисніть **Зберегти**.

Після цього всі нові повідомлення будуть надсилатися від вашого нового імені.

## Зміна теми

Натискайте кнопку **🌓 Тема**, щоб перемикатися між доступними темами оформлення.

## Збереження історії чату

Натисніть кнопку **💾 Зберегти чат**.

У папці з програмою буде створено файл:

text
chat_history.txt


У ньому буде збережена вся історія повідомлень.

## Очищення чату

Натисніть кнопку **🧹 Очистити чат**.

Чат очиститься лише у вашому вікні програми. У інших користувачів повідомлення залишаться.

## Статус підключення

🟢 **Онлайн** — успішне підключення до сервера.

🔴 **Відключено** — сервер недоступний або з'єднання втрачено.

## Можливі проблеми

### Не вдається підключитися до сервера

Переконайтеся, що сервер запущений.

### Повідомлення не надсилаються

Перевірте, чи працює сервер та чи є підключення.

### Не відкриваються зображення

Переконайтеся, що вибраний файл не пошкоджений.
